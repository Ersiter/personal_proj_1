01 · Overview

项目概览

业主戴启东，19岁大学生，与父母（父49岁、母52岁）及哥哥共同居住于佛山禅城。母亲患有帕金森综合症，室内设计需以安全、便利为核心底线，同时融入岭南文化元素。

设计起点

业主在《需求意向表》中写下的每一行字——从"广东人，你懂的"到"帕金森等老年综合症"，从"多岭南特色"到"一定要有一个茶几"——都被逐条解读。设计师的任务不是套模板，而是将这些口语化的愿望转译为精确的空间语言。

三大设计约束

  • 适老化底线 — 帕金森综合症母亲的安全是最高优先级
  • 岭南文脉 — "多岭南特色"需从建筑符号转化为现代空间语言
  • 务实预算 — 湛江普通家庭，拒绝过度装修
02 · Philosophy

设计理念

"现代极简骨架 + 岭南文化灵魂 + 宋式美学韵味"
线条克制、材料诚实、温度自然。

拒绝欧式雕花与传统中式的繁复堆砌。保留木作的温度，但用简化线条、现代材质重新演绎。岭南建筑符号——镬耳山墙、满洲窗、趟栊门——被"符号化"转译：可识别，但不复古；有根脉，但不陈旧。

设计不张扬，温暖不刻意。
老有所安——母亲可以安全地走完家里的每一寸路；
少有所乐——戴启东的手办和电动有属于它们的角落；
家有所依——一家人围坐吃饭，是最核心的空间定义。

03 · Lingnan Heritage

岭南文化 · 元素转译

六组岭南建筑符号，被提炼为现代室内空间的设计语言。

镬耳山墙

提炼弧形轮廓 → 天花造型装饰线、电视背景墙弧形收边、走廊灯槽曲线呼应。

满洲窗

彩色玻璃几何分割 → 玄关屏风、客餐厅柔性隔断、柜门装饰面板。

青砖灰瓦

灰调哑光砖 → 玄关 / 厨卫地面；仿古肌理涂料 → 局部墙面；青灰调贯穿硬装基底。

木雕花窗

简化几何镂空 → 空调出风口面板、装饰墙面、柜门通风隔断。

趟栊门

推拉隔断新演绎 → 客餐厅柔性分区，木格栅移门保持视线通透、空间可分可合。

庭院天井

阳台绿植 + 天光引入 → 微型室内景观，实木格栅天棚营造半室外感受。

04 · Elderly Care

适老化设计策略

以母亲（帕金森综合症）为核心使用者，构建全屋安全防护与便利体系。

安全防护

  • 全屋防滑地面 — 哑光砖防滑系数 R10+，无光滑表面
  • 连续扶手系统 — 走廊 + 卫生间 + 床边，管径 32mm
  • 门槛零高差 — 斜坡过渡 ≤ 5°，T 型铜条接缝
  • 卫生间安全 — 折叠沐浴凳 + L 型扶手 + 防滑垫
  • 感应起夜灯 — 床下 → 走廊 → 卫生间路径自动调暗
  • 紧急呼叫 — 床头 + 卫生间一键报警，联动子女手机

舒适便利

  • 宽通道 — 卧室门 ≥ 900mm，走廊 ≥ 1100mm（轮椅通行）
  • 杠杆门把手 — 省力易握，替代球形把手
  • 衣柜可升降 — 挂衣杆智能升降，减少踮脚弯腰
  • 厨房双高度台面 — 坐姿 / 站姿两用操作区
  • 语音控制 — 灯光 + 空调 + 窗帘，减少走动
  • 家具圆角 — 全屋无尖锐边角，沙发扶手加宽

针对帕金森综合症的特殊考量

高对比度色彩标识：门框、台阶、扶手使用与周围形成明显对比的颜色，帮助视觉定位。
减少精细操作：大面板开关（86 型）、感应水龙头、语音控制家电，杜绝细小旋钮与按键。
消除地面视觉干扰：统一素色哑光地面，避免图案纹理造成视觉错觉引发眩晕。
稳定支撑面：床离地 45cm 方便起坐，马桶旁设固定扶手，座椅带扶手支撑。

岭南天地（Lingnan Tiandi）是瑞安房地产「天地」系产品在佛山的延伸，与上海「新天地」同源。项目以祖庙东华里历史街区为文化底盘，将岭南骑楼、镬耳山墙、青砖灰瓦等元素与现代商住体量缝合，居住组团围绕街区展开。本户位于其中的高层住宅，南北通透，开间近 17 m，三面采光，是片区面积上限附近的大平层产品，适合三代同堂。

06 · Space Planning

空间功能分区

八室一厅，各安其位。每个空间的设计决策都来自意向表的原文对应。

客餐厅

71.20㎡
全屋最核心公共空间。6人L型沙发+茶几、75寸大屏电视。木饰面护墙板电视墙+隐形收纳。岭南花窗隔断柔性分区，暖色无主灯3000K。1×6实木餐桌+餐边柜+陈列柜，配独立电视。

主卧室（老年房）

28.61㎡
父母居住，适老化核心空间。2×2m实木床离地45cm，床头双侧扶手+紧急呼叫。可升降挂衣杆衣柜+圆角梳妆台。感应夜灯路径覆盖至主卫（9.66㎡），语音控制窗帘与灯光。

次卧一（戴启东）

24.43㎡
戴启东本人房间。双人床+极宽电脑桌+衣柜+书柜。手办展示区+电动娱乐角。耐脏墙面处理，暖色系+木地板。

次卧二（哥哥）

18.24㎡
哥哥房间。双人床+电脑桌+衣柜。简约实用风格，暖色系墙面+木地板。

次卧三（书房）

18.02㎡
改造为书房，满足2人同时使用。双人书桌+电脑位，书籍+手办展示柜。暖色系+实木地板。

厨房

11.49㎡
中式大吸力烟灶应对大油烟。U型操作台+双水槽，实用优先。暖木色橱柜面板+隐形拉手，易清洁哑光大砖。

阳台

5.79㎡
铝合金封阳台，实木格栅天棚。晒衣+储物功能，搭配琴叶榕/龟背竹微型景观。半室外岭南天井氛围。

卫生间 ×3

9.66 / 4.88 / 3.98㎡
三卫配置：主卫连接主卧，适老化全套扶手+折叠沐浴凳+智能马桶+紧急呼叫。次卫一淋浴房+台盆柜。次卫二实用为主。
